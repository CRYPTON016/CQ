This Project is about Deploying a Resume in JSON format in web

Used Tools and technologies are Firestore a Nosql database to store the relevant data, Cloud Function and Cloud Run for deployment and Python for writing the source

Steps

  • Create a New Project in GCP or use the old one if you got.
  • Go to Firestore and create collection - table and add the data in it
  • Configure a 2nd gen Cloud Function for deployment with a source that retrives data from Firestore
  • Check if you have Cloud Run working properly so cloud function can run on it
  • Make sure you provide required permissions for the service accounts so they can access the respective data properly.
  • Set the permission in cloud function for allUser so that everyone with the URL can access it
